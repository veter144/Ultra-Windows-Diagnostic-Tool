# Ultra Windows Diagnostic Tool

**Ultra Windows Diagnostic Tool** is a lightweight yet powerful diagnostic utility designed for the Windows operating system. It provides essential system information and diagnostic capabilities in a simple, user-friendly interface.

## 📦 Contents

This repository contains a precompiled, ready-to-use version of the application. No additional setup or compilation is required.

### File List

| File | Description |
|------|-------------|
| `ConsoleApp2.exe` | Main executable file |
| `ConsoleApp2.dll` | Core application library |
| `ConsoleApp2.deps.json` | Dependency manifest |
| `ConsoleApp2.runtimeconfig.json` | .NET runtime configuration |
| `Ultra Windows Diagnostic Tool (Safe).cmd` | Batch script for safe execution (recommended) |

## 🚀 How to Run

1. Download all files from the repository to a single folder on your computer.
2. Launch the tool using one of the following methods:

**Recommended method:**  
Double-click `Ultra Windows Diagnostic Tool (Safe).cmd`

**Alternative method:**  
Run `ConsoleApp2.exe` directly from the command line or File Explorer.

> ⚠️ **Note:** The batch file ensures proper environment settings and helps avoid potential execution issues.
